=head1 NAME

rum_runner kill - Kill a job running on a cluster.

=head1 SYNOPSIS

rum_runner kill -o dir

=head1 DESCRIPTION

Kill a RUM job running on a cluster. Note that to kill a job running
locally, you can simply run C<kill pid> where pid is the process id of
the parent rum_runner process. You can also kill it with CTRL-C if you
have it running in a terminal.

=head1 OPTIONS

=over 4

=item B<-o>, B<--output> I<dir>

Specify the output directory of the job. 

=back
